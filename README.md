## Getting Started


To get a local copy up and running follow these simple example steps.

### Prerequisites
- node js
- visual studio 
- webpack  

### Setup
To get a copy of this project follow the steps below
```
git@github.com:Tchilo/leaderboard.git
cd leaderboard
```


### Usage
in the terminal you can run the code below
```
npm start
```
